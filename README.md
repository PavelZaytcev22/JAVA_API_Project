# 📄 Умный принтер с мобильным управлением

## 🎯 Цель проекта
Создание системы беспроводной печати, включающей:
- Умный принтер, подключенный к серверу (Raspberry Pi/виртуальная машина)
- Мобильное приложение для Android с управлением печатью и предпросмотром документов

## 🏗 Архитектура системы

### 1. Сервер (Raspberry Pi / VM)
**Центральный узел для обработки запросов на печать**

**Функции:**
- 📥 Приём файлов от мобильного приложения (по Wi-Fi или через интернет)
- 🗂 Управление очередью печати (добавление/удаление задач)
- 🖨 Отправка заданий на подключенный принтер
- 📊 Предоставление информации о состоянии системы (статус, очередь, подключения)

### 2. Умный принтер
- Обычный USB-принтер с поддержкой стандартных протоколов печати
- Подключение: USB к серверному устройству

### 3. Мобильное приложение (Android, Java)
**Функции:**
- 📂 Выбор файла для печати из локального хранилища
- 📤 Отправка файлов на сервер печати
- 🗃 Просмотр и управление очередью печати
- 📡 Мониторинг состояния принтера и системы
- 👀 Предпросмотр документов перед печатью
- ⚙️ Настройки подключения и параметров печати

## ⚙️ Функциональные требования
- Беспроводное подключение (Wi-Fi/LAN)
- Поддержка форматов: PDF, JPEG, PNG, TXT, DOC/DOCX
- Управление очередью печати (добавление, приоритизация, отмена)
- Отображение статуса принтера и текущих заданий
- Просмотр подключенных устройств и их состояния
- Предпросмотр документов (для поддерживаемых форматов)

## 🔧 Технические требования
- **Серверная ОС:** Raspberry Pi OS/Linux
- **Язык разработки сервера:** Python 3.x
- **Протокол обмена данными:** REST API
- **Библиотеки для предпросмотра:** Android PDFRenderer, MuPDF
- **Система печати:** CUPS (Common UNIX Printing System)
- **Мобильная платформа:** Android 8.0+
- **Язык разработки:** Java/Kotlin

## 🚀 Этапы разработки
1. **Настройка серверной инфраструктуры**
   - Установка и настройка ОС на Raspberry Pi/VM
   - Конфигурация CUPS и подключение принтера
   - Настройка сетевого доступа

2. **Разработка серверного API**
   - Проектирование REST API endpoints
   - Реализация обработки файлов и очереди печати
   - Интеграция с системой печати CUPS

3. **Создание Android-приложения**
   - Разработка пользовательского интерфейса
   - Реализация функционала предпросмотра документов
   - Интеграция с серверным API

4. **Тестирование и отладка**
   - Функциональное тестирование компонентов
   - Интеграционное тестирование системы
   - Исправление ошибок и оптимизация
